Pumpkin Lands

Pumpkin Lands is a public modded server ran by mounderfod, on 1.18 Quilt.

How do I download it?

  1. Download and install Java 17 from here.
  2. Download and install PolyMC from here.
  3. Once PolyMC is installed, run it and go through the setup steps, making sure to select the JDK you just installed when prompted with Java settings.
  4. Download this ZIP file and drag it into the PolyMC window. Click OK.
  5. The recommended 5GB RAM has already been set but if you need to change it go to Edit Instance -> Settings and change the values there.
  6. The Java version should be 17. If the Java version selected is wrong then tick the checkbox next to Java installation, click Auto-detect and select the right one.
  7. Hit Launch. When the server is loaded, press the Connect button!
